# Listicle / roundup outreach — Clawmart

Draft template + target list for Ryan to send **manually**. Claude does not email anyone.
These are authors/editors of live "best AI visibility / GEO tools 2026" roundups — the
kind of pages that themselves rank in AI answers, so an inclusion is both a backlink and
AEO fuel. Verify the current byline on each page before sending (roundups get reassigned).

## Realistic expectations
Two kinds of targets here:
- **Neutral editorial** (SitePoint, TechnologyAdvice, Position Digital, ThatMarketingBuddy,
  GrowthOS, Amplitude, Dageno, Nightwatch) — genuinely add tools; **easiest wins.**
- **Competitor-owned blogs** (Profound, Frase, SE Ranking, Omnia, MyBrandi) — they publish
  roundups that include rivals for SEO, but adding a direct competitor is a harder ask.
  Pitch these as *complementary* (the one-time-fix / SMB slot they don't fill), not as a
  replacement. Lower hit rate; still worth a polite ask.

Lead with the **free tool + methodology transparency**, never with "please add me." Give
them a reason their readers benefit. Offer a free Fix Kit so they can test it firsthand —
honest reviewers convert far better when they've actually run the thing.

## Follow-up cadence
- One personalized send. If no reply in ~7 days, one short bump ("adding the free-kit
  offer in case it's useful — no worries if not").
- After the PH launch, re-send to non-repliers with the update ("we launched on Product
  Hunt / shipped N kits") — new information justifies a fresh touch. Two touches max, then
  drop it. Don't become the annoying tool founder in their inbox.
